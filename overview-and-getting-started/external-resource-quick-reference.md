# External Resource Quick Reference

This file is a quick-reference summary of all external documents. Use it to quickly identify a document based on its name or a user's query.

## Integration Manuals

-   **SimTransManual.pdf:** The complete user manual for SimTrans, the automation bridge between SigmaNEST and SM3. (229 pages)
-   **KastoDocs.pdf:** Technical documentation for integrating Kasto automated storage and sawing systems.
-   **ToyotaStockManagementSystem.pdf:** Guide for the Toyota ASRS (Automated Storage and Retrieval System) integration.
-   **SigmaNESTcontent.docx:** Additional content and details for the SigmaNEST integration.

## Labeling and Printing

-   **InventoryLabelFormats.pdf:** A visual catalog of all standard inventory label formats and templates.
-   **ShippingLabelFormats.pdf:** A visual catalog of all standard shipping label formats and templates.
-   **OverlayedMTR.pdf:** An example of a Mill Test Report (MTR) with an overlay.

## eCommerce

-   **SM3eCommercePortalUserGuide.pdf:** The complete user guide for the customer-facing eCommerce portal. Covers creating quotes, viewing orders, and accessing invoices. (13 pages)

## Banking and Payments (EFT/ACH)

-   **ACH_Auth.docx:** A generic authorization form for setting up ACH payments.
-   **Vendor_EFT_Auth.docx:** A generic authorization form for setting up EFT payments for vendors.
-   **EFT_CKV-*.docx:** A series of bank-specific EFT setup forms for various banks (ARP, FNB, HUN, KEY, WFG).

## Templates and Tools

-   **InventoryCountSheet.xls:** An Excel spreadsheet template to be used for conducting a physical inventory count.
-   **Barcode_Scanner_Printer.txt:** A text file listing the specific hardware requirements and recommended models for barcode scanners and printers compatible with SM3.

## Reference Documents

-   **AllReports.docx:** A comprehensive document that lists and describes all available reports within 4GL.
-   **IntercompanySales.docx:** A document detailing the process and configuration for intercompany sales transactions.
