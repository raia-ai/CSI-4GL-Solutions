IN74 - Transactions In A Date Range
===================================

This function reports on inventory transactions within a date range for a specified type of entry.

1. Choose Inventory Control » Inventory Reports » Transactions In a Date Range [IN74].
3. Enter or select the Product Line (by default, all are selected).
4. Enter or select the Type of Entry (by default, all are selected).
5. Indicate if the Run Type should extract data from the Beginning date or the End date.
6. Enter the Date From  and Date To  (To Date defaults to today's date).
