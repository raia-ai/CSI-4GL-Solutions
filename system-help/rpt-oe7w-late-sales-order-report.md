OE7W - Late Sales Order Report
==============================

This function generates a report listing all late (requested date to ship is prior to today's date) sales orders by customers.

1. Choose Sales Order Entry » Order Entry Reports » Late Sales Order Report [OE7W].
3. Enter or select a Customer code (by default, all are selected).
5. Type the minimum number of Days Late an order must be to be included.
6. Indicate if you want the report to be Sorted by Customer.
