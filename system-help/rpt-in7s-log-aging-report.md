# IN7S - Log Aging

This function produces a report to perform a price adjustment on product line(s). This report is also the first step in importing [Inventory Revaluation](https://4glsol.com/sm3-helpdocs/Content/Admin/Config/FM_FL_Util/Util_IN_InvRevaluation_IN10S.htm) (IN10S).

This report is only available as an Excel download.

1. Choose Inventory Control » Inventory Reports » Log Aging \[IN7S].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Click in the Product Line field to open the product line window.
4. Enter S to select or D to deselect product lines.
5. Enter the Alloy Type and press Enter.
6. Select at least one Product Line.
7. Press Enter to process the report. This creates an Excel File with pricing for the selected product lines.

**NOTE:** If you are generating this report for Inventory Revaluation Import, the Excel output must be modified; refer to [Inventory Revaluation Import](https://4glsol.com/sm3-helpdocs/Content/Admin/Config/FM_FL_Util/Util_IN_InvRevaluation_IN10S.htm)
