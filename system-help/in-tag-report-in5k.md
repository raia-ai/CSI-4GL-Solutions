Tag Report
==========

This report displays either all missing (not counted) tags, or all counted tags (tallies). You can run this report multiple times during the count.

1. Choose Inventory Report » Physical Inventory » Tag Report [IN5K].
3. Select the Report Type: Missing or Used/voided tags.
4. If you entered “U”, you can select one Count Operator to report on (or leave as ALL); if you selected ALL operators, indicate if the report should be Group by Operator.
