# IN7P - Products With Adders

This function produces a report of selected products with adders assigned.

1. Choose Inventory Control » Inventory Reports » Custom Reports » Products With Adders \[IN7P].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select a Product Line and Inventory Category (by default, all are selected).
4. Indicate if you want to show products with inventory On Hand .
5. Select how/where the report should be output (click … to choose from a list). When you press Enter the report will generate.
