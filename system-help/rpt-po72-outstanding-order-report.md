# PO72 - Open PO By Product

This function displays open purchase orders by delivery date for selected (or all) warehouses/vendors for a specified delivery date time period.

1. Choose Purchasing » PO Reports » Open PO By Product \[PO72].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Optionally select one or more product lines to filter the report.
4. Select a Vendor.
5. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
6. Indicate if you want to show the customer name on report. If “Y” is selected, sort/currency type/hide value fields are set at default and cannot be changed.
7. Select how you want to Sort the report: by Product code or by purchase order Number.
8. Indicate the currency type: Functional (currency assigned to warehouse) or Source (customer’s currency).
9. Indicate if you want to Hide $ Values on the report.
10. Indicate if you want to Include Receipts that are on hold.
11. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
