# IN7ZR - Inventory Cost/Avg Sales Price Analysis

This function produces an Inventory Cost/Avg Sales Price Analysis.

1. Choose **Miscellaneous » IN Functions » Inventory Cost/Avg Sales Price Analysis** \[IN7ZR].
2. Enter or select _Product Line_ code (by default, all are selected).
3. Leave default to exclude product groups with _Zero_ quantity, type “Y” to include them.
