# OE7ZK - Monthly Sales By Customer By Postal

This report displays monthly sales and gross margin percentage by customer.

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Custom Reports » Monthly Sales By Customer By Postal \[OE7ZK].
2. Enter or select the End Day of Month (defaults to last day of current month) - the report will display invoiced sales information for the 12 months ending with this date.
3. Enter or select a Customer Start Date (customers with a start date after this date will be included) or leave blank to include all customers.
4. If a territory is selected, Province and Operator fields will default to ALL.
5. Enter or select a State/Province (defaults to all). If a province/state is selected, Territory and Operator fields will default to ALL.
6. Indicate whether to create report by Salesperson, Outside salesperson, or order Writer.
7. Enter or select an Operator (defaults to all).
8. Indicate whether you want to view data for the Customer Salesperson (on the customer file); or leave blank to reference the salesperson on the invoice.
9. Indicate if you want to Group by Operator.
10. Specify whether to sort the data by Customer name, descending Sales, or Postal/zip code.
11. Specify whether to show the Bill to or Ship to address.
12. Indicate if you want to include customers with Zero Sales in the specified time period.
13. Indicate if you want to Show Details for each customer (contact, phone #, postal code, etc.).
14. Indicate if you want to Include Inactive customers (zero sales transactions in the past year).
