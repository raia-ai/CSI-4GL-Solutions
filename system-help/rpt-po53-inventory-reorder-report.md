# PO53 - Re-Order Report

1. Choose Purchasing » PO Reports » Re-Order Report \[PO53].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select Product Lines.
4. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
5. Enter Number of Months back to include on the report (defaults to 12).
6. Enter or select Sort Parameter: Product, Turn, Quantity, M = Onhand or ICC.
7. Indicate if you want to Show Inactive (products with no activity in the specified date range).
8. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
