Check Inquiry
=============

This function is used to inquire about vouchers assigned to check.

1. Choose Accounts Payable » AP Inquiries » Check Inquiry [AP67].
2. Enter or select Check Number.

   1. Enter or select Bank Account.
   2. Enter or select Check Number.
   4. Enter or select Vendor code, select check-box to choose all.
   5. Select Status: All, Returned, Void, Cleared, or Printed.
3. Press F4 to Add/View Remarks.
