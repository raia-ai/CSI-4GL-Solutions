PO7M - Corporate Re-Order Report
================================

This function is used to display purchases by product displaying previous/current month and 12 future months.

1. Choose Purchasing » PO Orders » Corporate Re-Order Report [PO7M].
3. Enter or select Product Line code (defaults to all).
4. Indicate if you want to Include Incoming BT.
