PR7M - Open IP/OP Report
========================

[need description]

This report is available as an Excel download only.

1. Choose In-House Production » In-House Production Reports » Open IP/OP Report [PR7M].
3. Enter or select the Type of Entry code (defaults to all).
4. Enter or select the Process code (defaults to all).
5. Enter or select the Machine code (defaults to all).
