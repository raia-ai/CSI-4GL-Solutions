# IN7ZD - Products With Zero Available

This function is used to display products with zero inventory available, incoming commitments for product and month turns for the last six months (182 days).

1. Choose Miscellaneous » IN Functions » Products with Zero Available \[IN7ZD].
2. Enter or select Product Line code (defaults to all).
