# IN7K - Stock List By Finish/Mill

This function produces an Inventory stock list by finish, mill price.

1. Choose Inventory » Inventory Reports » Stock List By Finish/Mill \[IN7K].
2. The report will be produced at the Product level; enter “L” to filter by Log level instead.
3. Indicate if you want to show products with inventory On Hand .
4. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
5. Enter or select Product Line code (by default, all are selected).
6. Enter or select the Grade (defaults to all).
7. Enter or select Finish (defaults to all).
8. Mill code (defaults to all).
9. Indicate if you want to Show Price/Value on report (by default, this is not selected).
10. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
