Direct Check
------------

This function is used to create a check and print without any extra steps. This is a quick way to get a check out.

1. Choose Accounts Payable » Processing Disbursements » Direct Check [AP21B\_B].
2. This process follows the same steps as Vendor Invoice Entry [AP21B] which can be found in Accounts Payable » Processing Vendor Invoices » Vendor Invoice Entry.
