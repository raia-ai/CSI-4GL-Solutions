# OE7XJ - Inventory Classification Report #2

1. Choose Sales Order Entry » Order Entry Reports » Inventory Classification Report #2 \[OE7XJ].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Select at least one Product Line.
4. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
5. Select how you want the report to Sort By: ICC, Product, Weight, Value, Margin, GM%, Onhand, Group 1-2-3, or Total number of transaction.
