Label Samples
=============

1. Update the source Word docs in \Words that Work Documentation\Other Docs\Sources for External Files in Help\
2. Save as PDF into the help/external files folder over the existing PDF
3. Send email to George, Paul, Rose, Denise, Sam:  
   “The label samples have been updated, PDFs available here:https://4glsol.com/sm3-helpdocs/Content/Common%20Functions/label\_samples.htm?Highlight=label%20sample”]

Labels are assigned to a particular label printer in IN19 > Label Options > Label Printer Defaults. If you need to set instructions for your label printer (encoding, shift (horizontal alignment) or top (vertical alignment) values), use MS35 > Label Opt.
