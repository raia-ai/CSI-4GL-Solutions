PR78 - Bill of Lading Re-Print
==============================

This function reprints a Bill of Lading.

1. Choose Miscellaneous » PR Functions » Bill of Lading Re-Print [PR78].
2. Enter or select Warehouse code (by default, Warehouse assigned to operator is selected).
3. Type of entry (display only).
4. Enter or select a Document code.
5. The associated Document Number (display only).
