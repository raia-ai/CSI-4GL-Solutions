Returned Check Entry
====================

This function is used to record a returned check entry in the general ledger.

1. Choose Accounts Payable » Check Reconciliation » Returned Check Entry [AP61].
2. Enter or select Check Number.
3. Enter Check Status. Enter “1” if returned or “^” to blank (outstanding).
4. Enter or select Cleared Date.
