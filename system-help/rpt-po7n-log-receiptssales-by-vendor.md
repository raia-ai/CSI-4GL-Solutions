# PO7N - Receipts/Sales By Log By Vendor

This function is used to display all log receipts by warehouse, by vendor, and by log, and provide selling information if the log has been sold.

1. Choose Purchasing » PO Reports » Receipts/Sales By Log By Vendor \[PO7N].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Select a Vendor.
4. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
5. Indicate if you want to Include Pre-Receipts.
6. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
