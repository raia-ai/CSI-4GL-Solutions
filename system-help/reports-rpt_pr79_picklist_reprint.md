PR79 - Picklist Re-Print
========================

This function reprints a picklist.

1. Choose Miscellaneous » PR Functions » Picklist Re-Print [PR79].
2. Enter or select Warehouse code (by default, Warehouse assigned to operator is chosen).
3. Enter or select the Type Of Entry.
4. Enter or select Document Number.
