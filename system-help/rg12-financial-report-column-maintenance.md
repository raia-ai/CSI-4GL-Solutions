Maintaining a List of Fields
============================

This function is used to create a list of fields, along with their corresponding description and column heading, that can be referenced when you create a custom financial report.

Call 4GL if you have any questions.

1. Choose General Ledger » Financial Report Writer » Maintaining a List of Fields [RG12].
2. Enter or select a source Field Code, up to four alpha-numeric characters.
3. Enter a Field Description for the column.
4. Enter a Column Heading (two lines).
