# PP74 - Open OP Orders By Vendor

This report displays all open Outside Processing and Inside Processing orders for a specified vendor or type of entry.

1. Choose Outside Processing » OP Reports » Open OP Orders By Vendor \[PP74].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select Type of Entry (defaults to all).
4. Enter or select Vendor code (defaults to all).
5. Select to Sort By document Number or due Date.
6. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
7. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
