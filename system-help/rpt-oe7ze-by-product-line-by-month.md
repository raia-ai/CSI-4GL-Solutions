# OE7ZE - By Product Line By Month

This report displays the monthly sales by product line.

1. Choose Sales Order Entry » Order Entry Reports » By Product Line By Month \[OE7ZE].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select the End Day of Month (defaults to last day of current month) - the report will display invoiced sales information for the 12 months ending with this date.
4. Enter or select the Territory code (defaults to all).
5. Enter or select Product Lines (defaults to all).
6. Indicate whether to Group by Operator according to the selected Operator Type (Salesperson, Outside salesperson, or order Writer).
7. Indicate whether to use the Customer Salesperson, or leave blank to use the salesperson on the invoices.
8. Specify to Sort data by Product line or Ascending sales.
9. Indicate whether to include product lines with Zero Sales.
10. Select whether to produce a Detailed or Summary report.
11. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
