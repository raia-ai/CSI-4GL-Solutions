OE7Z - Sales Order Report
=========================

This function generates a report listing all sales orders sorted by salesperson.

1. Choose Sales Order Entry » Order Entry Reports » Sales Order Report [OE7Z].

4. Enter or select the Document Status to include (by default, all are selected).
6. Indicate if you want to display sales Values in the report.
