# AR7R - Detailed AR Aging Report

This function generates a detailed report based on money owed by customers and aged according to the document date entered. A list of open documents for each customer will also be provided.

1. Choose Accounts Receivable » AR Reports » AR Aging Report \[AR7R].
2. Enter or select Warehouse code (defaults to all warehouses).
3. Enter or select the Territory code (defaults to all).
4. Enter or select the Customer code (defaults to ALL).
5. Select whether to produce a Detailed or Summary report.
6. In the Open/All Documents field, indicate if you want to include All documents or only documents in the Customer AR Credit file that have a status of Open (do not have monies applied against them).
7. Select the Aging Date to include (defaults to today's date).
8. If necessary, change the number of Aging Days for each period.
9. If you only want to include documents a certain number of days past due, enter the number of Past Due Days .
10. Indicate the currency type: Functional (currency assigned to warehouse) or Source (customer’s currency).
11. Indicate if you want aging calculated by Invoice or Due date.
12. Select whether to Sort By Currency, Salesperson or Territory.\
    If you entered “S”, select the Salesperson Type to include (Inside or Outside).
13. Indicate if you want to show Customer Details and Remarks.
14. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
