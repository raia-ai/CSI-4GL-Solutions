Branch Transfer Picking Confirmation
====================================

This function is used to edit branch transfer documents to transfer material within company warehouse locations.

1. Choose Inventory Control » Branch Transfer » Branch Transfer Picking Confirmation [IN42].
2. Enter Document Number.
3. Enter or select bill to Customer (defaults to all).
4. Enter or select Salesperson (defaults to all).
5. Select a row the next menus are long and complex not sure where to go from here.
6. same as sales order/ purchase order. Difference is warehouse. Type of entry needs to Be selected. Product code needs to exist in both warehouses. Code will give you an error.
