# IN7YB - Stock List By Customer

This function creates a report to display log numbers assigned to customers. The Excel output includes log remarks.

1. Choose Inventory Control » Inventory Reports » Stock List By Customer \[IN7YB].
2. Enter or select the Product code (by default, all are selected).
3. Indicate if you want to show products with inventory On Hand .
4. Indicate if you want to show products with inventory On Hand .
5. Indicate if you want to Show Program (defaults to Y).
6. Indicate if you want to show inventory in Quarantine.
7. Indicate if you want to Show Zero logs (logs/products with zero quantity); by default, they are not shown.
8. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
9. Enter or select the Customer code (defaults to ALL).
10. If the customer has multiple Ship To locations, select the one you want to report on.
11. Enter or select Product Line and Spec Type (by default, all are selected).
12. Enter or select Specification code; this defaults to “ALL” if the Spec Type = “ALL”.
13. Select how/where the report should be output (click … to choose from a list). When you press Enter the report will generate.
