# OE7YE - Sales By Customer By Product

This function produces a report which displays sales grouped by customer and product year-to-date for current fiscal year.

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Misc Reports » Sales By Customer By Product \[OE7YE].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Enter or select the Customer code (defaults to ALL).
4. Enter or select the Product Line.
5. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
