# OE7YG - Sales By Cust By Product Line Two Year Usage

This function generates a report which displays sales grouped by customer and product line for two years.

1. Choose Sales Order Entry » Order Entry Reports » Sales By Cust By Pline Two Year Usage \[OE7YG].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select End Day of Month (defaults to last day of current month).
4. Enter or select a Customer (defaults to all).
5. Select one or more Salesperson and enter the salesperson type (Inside or Outside).
6. Select one or more Product Line.
7. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
