Refreshing Report Parameters (n/a for Oracle)
=============================================

This function will refresh all the record counters for a custom financial report’s parameters.

1. Choose **General Ledger » Financial Report Writer » Report Parameter Refresh** [RG91].
2. Enter or select the Report Number or “ALL”.
