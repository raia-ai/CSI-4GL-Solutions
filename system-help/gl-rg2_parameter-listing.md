Printing a Report’s Parameters
==============================

This prints the report parameters for a custom report created in the Financial Report Writer (in RG1).

1. Choose General Ledger » Financial Report Writer » Parameter Listing [RG2].
2. Enter or select Report Number, or “ALL”.
