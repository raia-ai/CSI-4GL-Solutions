# IN7YT - Offcut Stock Listing

This report displays the same columns as in [IN71](https://4glsol.com/sm3-helpdocs/Content/Reports/rpt_IN7I_MTR_Prnt_Log_Number.htm) by log and [IN7ZG](https://4glsol.com/sm3-helpdocs/Content/Reports/rpt_IN7ZG_Log_Reservation_Report.htm), but only displays reservations of logs from cut products.

1. Choose Inventory Control » Inventory Reports » Custom Reports » Offcut Stock Listing \[IN7YT].
2. Enter or select Product code (by default, all are selected).
3. Indicate if you want to show products with inventory On Hand .
4. Indicate if you want to Show Program (defaults to Y).
5. Indicate if you want to show inventory in Quarantine.
6. Select the Log Types to display: Reserved logs, logs Not reserved, or  All.
7. Enter or select the Product Line (by default, all are selected) and optionally, any further specifications.
8. Select how/where the report should be output (click … to choose from a list). When you press Enter the report will generate.
