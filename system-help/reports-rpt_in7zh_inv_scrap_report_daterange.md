IN7ZH - Scrap Analysis by Date
==============================

This function is used to create an Inventory Scrap Report for a specified date range.

1. Choose Inventory Control » Inventory Reports Inventory » Scrap Analysis by Date [IN7ZH].
3. Enter or select a Product Line (by default, all are selected).
4. Enter the Scrap Type to filter report by; Order, Zero, Track, Revalue, Undefined, All (default).
5. Indicate if you want to show products in the Scrap Bin only.
6. Enter or select the Date From and Date To (both default to today's date).
