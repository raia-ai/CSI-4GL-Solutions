IN7ZY - Inventory Restocking By Piece
=====================================

This function is used to create an inventory restocking report displaying on-hand piece inventory. Summarizes product totals regardless of specification.

1. Choose Inventory Control » Inventory Reports » Custom Reports » Inventory Restocking By Piece [IN7ZY].
3. Enter or select Product Line code (defaults to all).
4. Indicate if you want to include Insufficient Only. [meaning?]
5. Indicate if you want to include  Bundles. [need definition?]
