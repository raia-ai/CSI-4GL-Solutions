Log Split Scrap Report/Clear
============================

This function could use a description.

1. Choose Inventory Control » Inventory Adjustment » Log Split Scrap Report/Clear [IN38].
3. Enter or select Product Line.
4. Enter or select Log Number (defaults to all).
5. Select between Clear or Unclear.
7. Enter “Y” for ok and enter “/” to cancel.
