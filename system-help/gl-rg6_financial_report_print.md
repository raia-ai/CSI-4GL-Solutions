Printing Your Financial Reports
===============================

This screen is used to print reports defined in the Financial Report Writer. All financial reports can be exported to Excel using printer code XLS.

Use RG3 to print a list of all financial reports defined in the Financial Report Writer.

1. Choose General Ledger » GL Reports » Financial Report Writer » Financial Report Print [RG6].
2. Enter or select the Fiscal Period. You can choose any year.
3. Enter or select the Start and End Report number.
4. Enter the Number of Columns: 6 = six columns, 8 = eight columns (default).
