OE7ZC - Customer Salesman Analysis
==================================

This report displays customer sales by salesperson within a date range.

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Misc Reports » Customer Salesman Analysis [OE7ZC].
3. Enter or select Customer code (defaults to all).
5. Indicate if you want to generate a Detailed report (leave blank for a summary).
6. Indicate if you want to Include Credits/Debits notes in the report.
