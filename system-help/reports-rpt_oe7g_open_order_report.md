OE7G - Open Order Report
========================

This function produces a report listing all open sales orders and remaining quantities to be shipped.

1. Choose Sales Order Entry » Order Entry Reports » Open Order Report [OE7G].


5. Enter the Document Status to include (defaults to all).
7. Indicate if the date range should be based on the **O**rder date or the **R**equested date.
8. Indicate if cost lines should be included; if selected, the report cannot be sorted by Product.
9. Indicate if the report should be Sorted by Product, Work Order by Salesperson, Request Date, or Customer.
11. Enter or select a Product Line (by default, all are selected).
