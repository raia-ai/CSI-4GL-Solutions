# GL76 - Summary GL Activity

This function is used to print a Summary General Ledger Activity report.

1. Choose General Ledger » GL Reports » Summary GL Activity \[GL76].
2. Enter Warehouse number (by default, your assigned warehouse is selected).
3. Enter General Ledger Account code (by default, all are selected).
4. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
5. Enter or select Source code (by default, all are selected).
6. Enter “U” to display Unposted transactions (by default, Posted are displayed)
7. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
