IN7ZU - Summary Log Aging Report
================================

This function is used to create a Summary Log Aging report.

1. Choose **Miscellaneous » IN Functions » Summary Log Aging Report** [IN7ZU].
3. Enter or select a *Product Line* (by default, no product lines are selected).
