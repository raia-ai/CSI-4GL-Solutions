# Log Split

This function is used to view log split document details.

1. Choose Inventory Control » Inventory Inquiry » Log Split \[IN6C].
2. Enter From Date.
3. Enter or select Origin (defaults to all).
4. Enter or select Split Number (defaults to all).
5. Enter or select Doc Number.
