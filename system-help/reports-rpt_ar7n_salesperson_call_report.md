AR7N - Salesperson Call Report
==============================

Generate a report of sales call remarks.

1. Choose Sales Order Entry » Order Entry Reports » Salesperson Call Report [AR7N].

4. Enter or select Salesperson code along with Inside our Outside denotation.
6. Enter or select Category code: AR = accounts receivable, IS = inside sales, OS = outside sales.
7. Enter Priority: High, Medium, Low, or ALL.
