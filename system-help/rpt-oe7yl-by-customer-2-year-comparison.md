# OE7YL - Sales By Salesperson By Customer 2

This report shows a two-year comparison of sales by customer by salesperson for the same period of each year.

1. Choose Sales Order Entry » Order Entry Reports » Sales By Salesperson By Customer 2 \[OE7YL].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Enter or select the Customer code (defaults to ALL).
4. Indicate if you only want to show customers with activity in the specified date range.
5. Select one or more Salesperson and enter the salesperson type (Inside or Outside).
6. In the Date To field, enter the end date of the most recent year period that you want to view. The report will show sales numbers for the year of the Date To and the same period of the previous year. Both year's sales numbers start with January 1 of that year up to the month and day of the Date To. For example, if you enter 04/27/23, the report will show sales numbers for 01/01/23 – 04/27/23, and 01/01/22 – 04/27/22.
7. Select if you want the report to Sort By sales Value or gross Margin, and indicate if it should also Sort By Salesperson.
8. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
