# Processing a POS Credit

OLD PROCESS, UNPUBLISHED, KEPT IN PROJECT FOR POSTERITY (FOR NOW)

To create a credit note for the full or partial amount of a POS invoice:

1. Choose Sales Order Entry » Point of Sale » POS Credit \[OE53].
2. Enter or select the POS Invoice Number.
3. Enter the Return Reason for the credit note.
4. Indicate if the log should be returned as the **S**ame log #, or if a new log # should be **G**enerated instead.
5. If necessary, change the Credit Note Date.
6. Click Process. The Credit Memo document opens.
7. Make any changes necessary; if you are processing a partial credit only, on the Details tab delete the lines that you do not wish to credit.
8. Click Process.
