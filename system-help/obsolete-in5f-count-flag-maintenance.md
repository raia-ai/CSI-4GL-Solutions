# Count Flag Maintenance

\[Oct/20: John couldn't identify the purpose of this function, said it's not used, ignore it. Leaving here for posterity because it's still in the menu]

1. Choose Inventory Control » Physical Inventory » Count Flag Maintenance \[IN5F].
2. Enter or select Log Number.
3. Select Counted Flag: Y = counted, C = updated, blank = not counted.
