# PO77 - Stock Usage and Re-Order

1. Choose Purchasing » PO Reports » Stock Usage and Re-Order \[PO77].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select Product Line (defaults to all).
4. In the Back Weeks field, enter the number of weeks to go back to start accumulating usage figures.
5. In the Back Months field, enter the number of months back to use to calculate the average monthly usage.
6. Enter the Display Type : products with Suggested quantity, Release quantity, Both, Either or All.
7. Indicate if you want to Sort By Available quantity.
8. If you entered “All” in the Warehouse field, indicate if you want to Group by Whse.
9. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
