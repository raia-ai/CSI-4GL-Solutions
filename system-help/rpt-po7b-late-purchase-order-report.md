# PO7B - Late Purchase Orders

This function is used to display late (by specified days late) purchase orders, by delivery date for selected warehouse, selected (or ALL) vendors for a specified delivery date time period and/or # of days late in ascending purchase order number.

1. Choose Purchasing » PO Reports » Late Purchase Orders \[PO7B].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select a Vendor Code (defaults to all).
4. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
5. Indicate if you want to Show Ontime receipts.
6. Indicate if you want to Show Early receipts. If you enter “Y”, enter the Min and Max Early Days.
7. Indicate if you want to Show Late receipts. If you enter “Y”, enter the Min and Max Late Days.
8. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
