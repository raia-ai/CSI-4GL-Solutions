IN7YC - External Product Code to Log Report
===========================================

This function displays the external product code and corresponding  product code.

This report is available as an Excel download only.

1. Choose Miscellaneous » IN Functions » External Produce Code to Log Report [IN7YC].
3. Enter “Y” to download the report or “/” to cancel it.
