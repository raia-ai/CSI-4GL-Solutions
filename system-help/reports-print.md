PP79 - OP Picking Slip
======================

This function is used to reprint a picklist.

1. Choose Outside Processing » OP Reprints » OP Picking Slip [PP79].
3. Enter or select Type Of Entry (defaults to processing purchase order, PP).
4. Enter or select Document Number.
