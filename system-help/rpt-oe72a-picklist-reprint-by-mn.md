# OE72A - Picklist Reprint by Manifest Run

This function reprints pick slips for a manifest run by sales order and bin location.

1. Choose Sales Order Entry » Order Entry Reprints » Picklist Reprint by Manifest Run \[OE72A].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Select the Manifest and Run ID.
4. If you want to print by machine, select the check box and select a machine or leave “ALL” machines.
5. If you want to print by bin location, select the check box and select a bin or leave “ALL” bins.
6. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
