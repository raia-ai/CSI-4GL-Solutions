# OE72A - Picklist Reprint by Manifest Run

This function reprints pick slips for a manifest run by sales order and bin location.

1. Choose Sales Order Entry » Order Entry Reprints » Picklist Reprint by Manifest Run \[OE72A].
2. Select the Manifest and Run ID.
3. If you want to print by machine, select the check box and select a machine or leave “ALL” machines.
4. If you want to print by bin location, select the check box and select a bin or leave “ALL” bins.
