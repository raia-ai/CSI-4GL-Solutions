Deleting Journal Entries [TO BE DELETED]
========================================

[GL37 was removed in 3.24; GL39 will be removed as well; when it is, delete this topic]

You can only delete journal entries that have not been posted. An authorization code is required;  to obtain this code.

1. To delete a regular journal entry, choose General Ledger » Journal Entry » Journal Delete [GL37].   
   or  
   To delete a recurring journal entry, choose General Ledger » Journal Entry » Recurring Journal Delete [GL39].
3. Press F3 to enable the OK prompt where you must enter the Authorization Code provided by 4GL Support.
