PO7D - Direct Receipts By Shipment Number
=========================================

This function is used to display shipment receipts.

1. Choose Purchasing » PO Reprints » Direct Receipts By Shipment Number [PO7D].
3. Enter or select the Document number.
4. Indicate if you want to Print Labels.
