# AR75 - Listing Customers

This function is used to print a report displaying customer accounts.

1. Choose Accounts Receivable » AR Reports » Customer Listing \[AR75].
2. Enter or select the Customer code (defaults to ALL).
3. Select Multi Territories.
4. Select the Inside Salesperson whose customers are to be included on the report (defaults to all).
5. Select the Outside Salesperson whose customers are to be included on the report (defaults to all).
6. Select how to sort the list: by customer Name or Postal/zip code (default).
7. Enter “Y” to Show Inactive only, leave blank to exclude them.
8. Specify whether to Show Details on the report: no (blank, default) or Yes.
9. Show whether to show the customer's Ship To address(es): no (blank, default) or Yes.
10. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
