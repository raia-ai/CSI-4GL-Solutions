# AR75 - Listing Customers

This function is used to print a report displaying customer accounts.

1. Choose Accounts Receivable » AR Reports » Customer Listing \[AR75].
2. Select Multi Territories.
3. Select how to sort the list: by customer Name or Postal/zip code (default).
4. Enter “Y” to Show Inactive only, leave blank to exclude them.
5. Specify whether to Show Details on the report: no (blank, default) or Yes.
6. Show whether to show the customer's Ship To address(es): no (blank, default) or Yes.
