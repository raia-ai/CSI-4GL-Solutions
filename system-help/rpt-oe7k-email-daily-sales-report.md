# OE7K - Daily Sales Report

This function generates and emails a daily sales report.

1. Choose Sales Order Entry » Order Entry Reports » Daily Sales Report \[OE7K].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Enter or select Date.
4. Specify the invoice types to run report for: SHP = all shipped but not invoiced, UPD = Invoiced sales by not yet paid, or ALL.\
   When you press Enter the report will be generated and emailed.
