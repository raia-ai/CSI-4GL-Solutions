OE7ZH - Sales Order Report By Date
==================================

The function produces a report listing all sales orders.

1. Choose Sales Order Entry » Order Entry Reports » Sales Order Report By Date [OE7ZH].
3. Enter or select Salesperson code (by default, all are selected).
4. Specify how to Sort the data: Customer, Document Date, Order Date, or Required date (default).
5. Indicate if you want to Group By Operator, or leave blank to group by salesperson.
