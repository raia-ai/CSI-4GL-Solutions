IN7ZE - Product Availability And Incoming Report
================================================

This function displays products available and incoming for a product or product line(s).

1. Choose Miscellaneous » IN Functions » Product Availability And Incoming Report [IN7ZE].
3. Enter or select Product code (by default, all are selected).
4. Enter or select Product Line code (by default, all are selected)
5. Incoming PO? (not included in any previous documents).
