Using Hard Allocations at the Log Level
=======================================

This setting enables salespeople to select and reserve a quantity against a log during order entry.

1. Choose Administration » System Controls [MS33].
2. In the Orders Entry section, click in the More Options 2 field, then double-click to open the Order Entry Options 2 screen.

![](../../Resources/Images/ADMIN_SysControls_OE_More2.png)

3. In the Allow Log Reservations? field, enter “R” to allow reservations at the log level.

![](../../Resources/Images/ADMIN_SysControls_OE_More_AllowLogRes.png)
