# Reversing Split Log Document

This function is used to reverse Log Split created in LULOGSP9. Cannot reverse if Split logs created have been sold, picked or have been committed to a document.

1. Choose Inventory Control » Inventory Adjustment » Split Log Document Reversal \[IN39].
2. Press F4 to view log Details.
3. Check box to have this piece returned to the Order.
4. Check to have this piece returned to original Log number.
5. Is this piece a scrap remnant enter “Y” for yes or leave blank for no in Scr field.
6. Press F4 to Revalue this piece of inventory.
