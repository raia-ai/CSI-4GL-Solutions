Expanding Tag Allocation
========================

This function is used to increase the number of physical inventory count tags required, originally setup when the physical inventory count was initiated in IN5B.

The “Use Tags on Inventory Count?” switch [] must be set to “Yes”.

1. Choose Inventory Control » Physical Inventory » Expand Tag Allocation [IN5J].
3. The Start and End Tag Numbers, as defined in IN5B, are displayed. Enter the new End Tag Number.
4. Press Enter to process the tag update.
