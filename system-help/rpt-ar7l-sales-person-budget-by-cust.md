AR7L - Sales By Customer By Product Line Budget
===============================================

This function prints the Sales Budget by customer, as defined in the customer file.

1. Choose Sales Order Entry » Order Entry Reports » Salespsn By Cust By Pline Budget [AR7L].
3. Enter or select Salesperson code as well as Inside or Outside designation.
4. Select the Date for the report (by default, report includes data to the current date).
5. Choose to sort by Customer name, Target Percent Variance or actual Sales.
6. Indicate if you want to show  Target Only.
7. Specify if you want the Province Summary to be by Bill-to or Ship-to address.
8. Specify if you want the salesperson Budget to be by Weight, sales Value or **M**argin value.
