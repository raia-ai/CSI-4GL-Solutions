OE7Q - By Product With a Dimension
==================================

This function produces a report which displays the total weight and sales value for all sales within a specified date range, broken down by product by dimension. This report totals sales value and quantity for warehouse and Grand Total.

1. Choose Miscellaneous » OE Functions » By Product With a Dimension [OE7Q].

4. Enter or select Grade (defaults to all).
5. Enter or select Start and End Dimensions in Inches.
6. Enter or select Start Date and End Date (defaults to today's date).
