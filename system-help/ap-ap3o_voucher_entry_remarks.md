Entering Voucher Remarks
========================

This function is used to enter voucher remarks against a AP voucher which has been posted, but not yet paid.

1. Choose Accounts Payable » Vendor Invoice Processing » Voucher Entry Remarks [AP3O].
2. Enter or select Voucher number where remarks are to be added (optional if vendor is provided).
3. Enter or select Vendor code (mandatory if voucher number is blank).
4. Enter or select Invoice code.

   Press modify to change values in line.
5. Add remarks in + field.
