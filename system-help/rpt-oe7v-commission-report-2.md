# OE7V - Commission Report #2

This function prints an Inside salesperson commission report for invoices within a date range for invoices processed on screen OE4D.

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Misc Reports » Commission Report #2 \[OE7V].
2. Enter or select Warehouse code (defaults to all warehouses).
3. Enter or select a Salesperson code (defaults to all).
4. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
5. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
