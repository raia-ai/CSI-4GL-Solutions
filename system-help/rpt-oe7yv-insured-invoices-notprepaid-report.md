OE7YV - Insured Invoices Not Prepaid Report
===========================================

This function produces a report which list invoices, not including those were term “Prepaid”. Prepaid identified in screen Terms Master file.

1. Choose Miscellaneous » OE Functions » Insured Invoices Not Prepaid Report [OE7YV].
3. Enter or select Province/State (defaults to all).
4. Enter or select Type of Entry code (defaults to all).
5. Enter or select Inv Date From and Inv Date To date of display period.
6. Specify Invoice types you want to run the report for. **SHP** = All shipped but not invoiced. **UPD** = Invoiced sales but not yet paid. All = all invoices.
7. Filter invoices by **S**alesperson or Order **W**riter.
8. Sort by: Document **N**umber, **O**perator, **D**ocument date, view detail **L**ines.
9. Enter “Y” to have credit and debit memos referenced using the corresponding invoice date instead of credit/debit memo date (defaults to no).
