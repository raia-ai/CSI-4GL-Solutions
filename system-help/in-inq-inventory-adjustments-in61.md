Viewing Inventory Adjustments
=============================

This function is used to view inventory adjustments.

1. Choose Inventory Control » Inventory Inquiry » Inventory Adjustments [IN61].
3. Enter or select Type Of Entry code: AJ = adjustment entry, HH = handheld adjustment entry.
5. Enter or select Weight unit of measure.
