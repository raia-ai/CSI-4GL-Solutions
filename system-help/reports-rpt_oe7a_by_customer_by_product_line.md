OE7A - Sales By Salesperson By Customer By Product Line
=======================================================

This report includes the invoiced sales value, weight quantity and gross margin percentage for each product line by customer, within a given date range.

1. Choose Sales Order Entry » Order Entry Reports » Sales By Salespsn By Cust By Pline [OE7A].
3. Indicate if you want to sort data by Operator (default) or Territory.
4. If Sort Order=“O”, select if the report should then be grouped by **S**alesperson or Order **W**riter.   
   If Sort Order=“T”, select the Territories to be included (in the window, indicate if you are **S**electing or **D**eselecting territories.
5. Specify to create the report by Salesperson or Order Writer.
6. Enter or select Operator code (defaults to all).
7. Enter or select Customer number (defaults to all).
8. Enter or select Product Line code (defaults to all).
9. Enter the Dimension you want to report on.

12. Specify whether to create a Summary or Detailed (default) report.
