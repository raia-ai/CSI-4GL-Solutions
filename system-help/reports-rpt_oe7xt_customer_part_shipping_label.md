OE7XT - Customer Part Shipping Label
====================================

This function is used with custom-generated shipping labels. If you plan to use this function, please  to discuss your custom label requirements.

1. Choose Sales Order Entry » Order Entry Reprints » Customer Part Shipping Label [OE7XT].
3. Enter or select the Type of Entry, Document and  Line.
4. If the label settings1 for this label is greater than 1, enter the Quantity to print.
5. Enter the number of Labels to reprint.

Configuration


Label Options > Label Printer Defaults > SL04 > Num
