PR7B - In-house Production Machine Analysis Report
==================================================

This function produces a report displaying usage values by Machine.

1. Choose **Miscellaneous » PR Functions » In-House Production Machine Analysis Report** [PR7B].
2. Enter or select *Warehouse* code (by default, warehouse assigned to operator is selected).
3. Enter or select *Machine* code (by default, all are selected).
4. Enter or select *Date From* and *Date To* (by default, today's date is selected for both).
