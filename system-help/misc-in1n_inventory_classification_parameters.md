Inventory Classification Parameters
===================================

This function is used to view/update inventory classifications. Classifications can also be assigned to product codes. Up to three criteria. …

1. Choose Purchasing » Inventory Re-Order » Inventory Classification Parameters [IN1N].
3. Enter or select Inventory Classification code.
4. Enter Inventory Classification Description.
5. Enter Margin From and To.
6. Enter Number of Sales To and From.
7. Enter Weight To and From.
