OE7YT - Lost Quote Reason Report
================================

This function prints a report listing quotes by Lost Quote Reason code.

1. Choose Sales Order Entry » Order Entry Reports » Lost Quote Reason Report [OE7YT].

4. Enter or select Salesperson code (defaults to all).
5. Indicate if you want to Group by Operator code. If you leave this blank, the report will be sorted by quote reason.
6. Indicate if you want to Show Remarks.
7. Enter or select the Reason Code to report on (by default, all are selected).
