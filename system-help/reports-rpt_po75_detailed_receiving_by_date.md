PO75 - Receipts By Currency By Date
===================================

This function is used to display detailed receiving reports by selected date range, by vendor.

1. Choose Purchasing » PO Reports » Receipts By Currency By Date [PO75].
3. Enter or select the Type of Entry (defaults to all).
5. Enter or select Vendor code (defaults to all).
6. Select the Status (defaults to all).
