# OE7ZU - Product Line Summary

This report summarizes the available product lines.

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Custom Reports » Product Line Summary \[OE7ZU].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Select one or more Product Lines.
4. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
5. Indicate if you want to Group by Alloy type.
6. Specify how to Sort the data: by sales Value, Margin, or Gross margin percentage.
7. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
