AP7H - Vendor Invoice Listing by Account Type
=============================================

This function produces a list of total monies paid to vendors type, for current and previous fiscal years.

1. Choose Miscellaneous » AP Functions » Vendor Invoice Listing by Account Type [AP7H].

4. Enter or select Vendor Account Type (by default, all accounts are chosen).
5. Type “Y” to filter by active accounts (by default, the report is not filtered by active accounts).
