OE7ZL - Invoice Cost Analysis
=============================

This function generates an Excel file that provides invoice costs for a date range.

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Custom Reports » Invoice Cost Analysis [OE7ZL].
3. Enter or select Type of Entry: CM = credit memo, DM = debit memo, IV = invoice, or ALL.
4. When you press Enter, the report is generated and opens in Excel.
