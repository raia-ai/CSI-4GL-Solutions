# AR7I - Printing Document Applications

This function prints all Accounts Receivable documents, sorted by customer.

1. Choose Accounts Receivable » AR Reports » Document Applications \[AR7I].
2. Enter or select Warehouse code (defaults to all warehouses).
3. Enter or select the Customer code (defaults to ALL).
4. Enter or select Type of Entry code.
5. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
6. Enter “Y” to include Plus Balance or leave blank to exclude.
7. Enter “Y” to include Minus Balance or leave blank to exclude.
8. Enter “Y” to include Zero Balance amounts or leave blank to exclude.
9. Enter “Y” to show Applied Only (defaults to all).
10. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
