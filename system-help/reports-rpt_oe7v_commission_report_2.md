OE7V - Commission Report #2
===========================

This function prints an Inside salesperson commission report for invoices paid within a date range for invoices processed on screen OE4D (xref).

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Misc Reports » Commission Report #2 [OE7V].
3. Enter or select a Salesperson code (defaults to all).
