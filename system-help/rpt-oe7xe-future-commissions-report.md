OE7XE - Future Commissions Report
=================================

This function generates a report listing commission accruals for Invoices in AR not yet paid plus invoices with commission where commission has not yet been paid, sorted by outside salesperson code. (see report OE7XB). Will only list invoices where a commission rate has been assigned.

1. Choose Miscellaneous » OE Functions » Future Commissions Report [OE7XE].
3. Enter or select Salesperson (defaults to all).
4. Enter or select a Customer number (defaults to all).
