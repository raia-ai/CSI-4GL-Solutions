# OE7BA - Proforma Invoice

NOTE: This invoice may be printed automatically if a pre-invoice is required in the customer file, or if credit terms require it for COD orders.

1. Choose Sales Order Entry » Order Entry Reprints » Proforma Invoice \[OE7BA].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select the Document you want to reprint.
4. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
