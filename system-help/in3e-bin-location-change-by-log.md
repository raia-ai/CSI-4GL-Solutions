Changing Bin Location By Log
============================

This function could use a description.

1. Choose Inventory Control » Inventory Adjustment » Bin Location Change By Log [IN3E].
3. Enter or select Number. What is the number for?
4. Enter or select New Location: Floor, Kasto, Quarantine, Program or Other. Select Warehouse Location.
5. Enter New Remarks, up to three lines.
