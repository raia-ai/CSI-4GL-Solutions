IN7S - Log Aging
================

This function produces a report to perform a price adjustment on product line(s). This report is also the first step in importing Inventory Revaluation (IN10S).

1. Choose Inventory Control » Inventory Reports » Log Aging [IN7S].
3. Click in the Product Line field to open the product line window.
4. Enter **S** to select or **D** to deselect product lines.
5. Enter the Alloy Type and press Enter.
6. Select at least one Product Line.
7. This creates an Excel File with pricing for the selected product lines.

If you are generating this report for Inventory Revaluation Import, the Excel output must be modified; refer to Inventory Revaluation Import.
