# OE7G - Open Order Report

This function produces a report listing all open sales orders and remaining quantities to be shipped.

1. Choose Sales Order Entry » Order Entry Reports » Open Order Report \[OE7G].
2. Enter or select the Warehouse code and the warehouse type (Inventory or Sales).
3. Enter or select the Territory code (defaults to all).
4. Enter or select the Customer code (defaults to ALL).
5. Enter the Document Status to include (defaults to all).
6. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
7. Indicate if the date range should be based on the Order date or the Requested date.
8. Indicate if cost lines should be included; if selected, the report cannot be sorted by Product.
9. Indicate if the report should be Sorted by Product, Work Order by Salesperson, Request Date, or Customer.
10. Select one or more Salesperson and enter the salesperson type (Inside or Outside).
11. Enter or select a Product Line (by default, all are selected).
12. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
