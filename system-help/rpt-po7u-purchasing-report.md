# PO7U - Purchasing Report

This report is only available as an Excel download.

1. Choose Purchasing » PO Reports » Purchasing Custom Reports » Purchasing Report \[PO7U].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select Product Lines.
4. Enter or select Spec Type (defaults to all).
5. Enter or select the end Date.
6. Press Enter to process the report.
