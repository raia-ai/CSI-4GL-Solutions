# IN7YG - Product Availability

This function is used to create an Inventory Stock list, showing available quantities.

1. Choose Inventory Control » Inventory Reports » Product Availability \[IN7YG].
2. Leave default to report at Product level or select Log level or Grade level.
3. Enter or select Product Code (by default, all products are selected). .
4. Indicate if you want to show products with inventory On Hand .
5. Indicate if you want to Show Program (defaults to Y).
6. Indicate if you want to show inventory in Quarantine.
7. Indicate if you want to Show Zero logs (logs/products with zero quantity); by default, they are not shown.
8. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
9. Enter or select the Product Line.
10. Enter or select Spec Type (defaults to all).
11. Enter or select Specification code (defaults to all).
12. Indicate if you want to Sort by Length (sort logs by their size description).
13. Select how/where the report should be output (click … to choose from a list). When you press Enter the report will generate.
