OE7XP - Output Delivery Note Logs To Excel
==========================================

[need description]

1. Choose Sales Order Entry » Order Entry Reprints » Output Delivery Note Logs To Excel [OE7XP].
3. Enter or select the Type of Entry to include.
4. Enter or select the Document number.
5. Enter “Y” in OK to generate; the report opens in Excel.
