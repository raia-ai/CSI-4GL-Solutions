# OE7YO - Sales Analysis by Product Line Group

This report lists Year Sales analysis by product line groups. Data will only include “I” lines on invoice details.

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Custom Reports » Sales Analysis by Prod Line Group \[OE7YO].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Enter or select Product Line (defaults to all).
4. Enter the Year you want to report on.
5. Select whether to produce a Detailed or Summary report.
6. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
