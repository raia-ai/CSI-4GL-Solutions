AP77 - AP Aging Report
======================

This report displays money owed to vendor. It can be run for a single vendor or ALL, currency type, invoice date or effective date.

1. Choose Accounts Payable » AP Reports » AP Aging Report  [AP77].
3. Enter or select a Vendor (by default, all are selected).

6. Enter or select Vendor currency (by default, all currency is selected).
7. Enter or select Vendor category (by default, all are selected).
8. Enter Date Used: Effective date or Invoice date.
9. Select the Future Applications checkbox if you want to apply any post-dated payments to the open balance.
