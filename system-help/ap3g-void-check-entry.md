# Voiding a Check

This function is used to check and update the appropriate entries in the GL.

This function will void the check selected.

1. Choose Accounts Payable » Disbursements Processing » Void Check Entry \[AP3G].
2. Enter or select Check Number:
   1. Enter or select Bank Account.
   2. Enter or select Check Number.
   3. Enter or select Vendor or select checkbox for all.
   4. Select Status: All, Void, Printed, Returned and Cleared.
3. Enter or select Void Date.
