# Maintaining Log By Log number

This function is used to maintain product log information.

1. Choose Inventory Control » Inventory Adjustment » Log Maintenance By Lognumber \[IN1E].
2. Enter Log Number.
3. Enter or select Picked Quantity (committed out in IUM).
4. Enter or select inventory log Movements.
5. Enter Case information for log.
6. Enter Heat number.
7. Enter Location: Floor, Kasto, Quarantine, Program or Other along with warehouse location. Cant get past this field to see which are used in the next section...
8. Vendor Barcode - assigned in WF21 (Receiving) and used in WF11 (Picking Confirmation) to retrieve all received logs associated with the same vendor barcode.
