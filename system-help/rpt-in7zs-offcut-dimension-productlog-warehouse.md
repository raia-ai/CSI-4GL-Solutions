IN7ZS - Off-Cut Dimensions by Product/Log Number by Warehouse
=============================================================

This function produces off-cut dimensions by product/log number by warehouse.

1. Choose **Miscellaneous » IN Functions » Off Cut Dimensions by Product/Log Number by Warehouse** [IN7ZS].
3. Enter or select *Product Line* code (by default, all are selected).
