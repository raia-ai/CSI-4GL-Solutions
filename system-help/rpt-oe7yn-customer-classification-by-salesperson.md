# OE7YN - Customer Classification By Sale

This report lists all open sales orders and the remaining quantities to be shipped, by order writer.

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Custom Reports » Customer Classification By Sales \[OE7YN].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Select the Salesperson Type (Inside, Outside, or order Writer), then enter or select the Salesperson (defaults to all).
4. Enter or select a customer Classification (defaults to all).
5. Enter the To Date (defaults to today’s date).
6. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
