# OE7M - Open Order By Order Writer Report

This function is used to generate a report listing all open Work Orders and the remaining quantities to be shipped by order writer.

1. Choose Miscellaneous » OE Functions » Open Order By Order Writer Report \[OE7M].
2. Enter or select Territory code (defaults to all).
3. Sort by Order Writer or Territory.
4. Specify date to use for date range: Y = delivery date, Blank = order date.
5. Enter or select Order Writer code (defaults to all).
6. Specify whether to include sales value in report: Y = Include the values, Blank = Don't include the values.
