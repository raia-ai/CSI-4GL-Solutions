OE7M - Open Order By Order Writer Report
========================================

This function is used to generate a report listing all open Work Orders and the remaining quantities to be shipped by order writer.

1. Choose Miscellaneous » OE Functions » Open Order By Order Writer Report [OE7M].
3. Enter or select Territory code (defaults to all).
4. Sort by Order Writer or Territory.
5. Specify date to use for date range: Y = delivery date, Blank = order date.
7. Enter or select Order Writer code (defaults to all).
8. Specify whether to include sales value in report: Y = Include the values, Blank = Don't include the values.
