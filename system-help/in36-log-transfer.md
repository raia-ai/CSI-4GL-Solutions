Transferring Logs
=================

This function is used to transfer (whole or partial quantities) from an existing Log to a new log. This screen could be used to transfer a remnant piece from a flat panel product code to a Flat Panel cut x cut Product Code. This function could also be used if a bundle is broken into one or more bundles.

1. Choose Inventory Control » Inventory Adjustment » Log Transfer [IN36].
3. Enter source Log Number.
4. Enter Transfer Quantity not sure what the second box could be for.
5. Select Destination Product line and code.
