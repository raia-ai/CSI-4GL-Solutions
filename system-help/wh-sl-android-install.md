StayLinked Android Client Installation
======================================

StayLinked is software that is used with 's handheld devices to help with connectivity and support. The StayLinked Administrator app includes tools to assist in troubleshooting problems, fully remote control or observe a session, run diagnostics on a mobile device, collect log files, send text messages to a device, restart the client software, reboot the mobile device, change device configurations, execute commands, run programs, manage licenses and more.
