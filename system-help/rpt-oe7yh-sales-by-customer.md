# OE7YH - Sales By Salesperson By Customer

This report shows sales by customer or by salesperson.

1. Choose Sales Order Entry » Order Entry Reports » Sales By Salesperson By Customer \[OE7YH].
2. Select one or more Warehouses and then click Close.
3. Enter or select the Customer code (defaults to ALL).
4. Enter or select the Territory code (defaults to all).
5. Indicate if you want to only show customers with activity in the specified date range.
6. Enter or select Salesperson and filter by salesperson type: Inside, Outside, Order Writer. Indicate if the salesperson source is Customer or Order.
7. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
8. Select if you want the report to Sort By sales Value or gross Margin.
9. If you selected “ALL” in the Salesperson field, indicate if the report should Group All Salespeople. If selected, all data will be consolidated under one salesperson heading.
10. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
