# OE75 - Invoice/Credit or Debit Note

This function is used to reprint/fax/email a credit or debit memo.

1. Choose Sales Order Entry » Order Entry Reprints » Invoice/Credit or Debit Note \[OE75].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Select the Type of Entry, then enter or select a Single Document number or use the Multiple Document field to select more than one document.
4. Indicate if you want to Include POD (proof of delivery<sup>1</sup>) in a separate PDF from the invoice.
5. Indicate if you want to include the word “Reprint” in the top right corner of the reprint.
6. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.

[![Open](https://4glsol.com/sm3-helpdocs/Skins/Default/Stylesheets/Images/transparent.gif)Configuration](https://4glsol.com/sm3-helpdocs/Content/Reports/rpt_OE75_Invoice_CrDr_Memo.htm?Highlight=oe75)

<sup>1</sup> PODs must be scanned and stored in a network folder, similar to MTRs; the POD folder address is defined in IN19.
