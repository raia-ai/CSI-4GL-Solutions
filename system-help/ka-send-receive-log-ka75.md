Send/Receive Log [KA75]
=======================

This inquiry displays the net result of a transaction. The system will match up a HLAU Telegram with the corresponding LHAU Telegram. The information is captured in a repository called KAD\_DTL. It will show what was requested by the system and what was returned by Kasto.
