Locating a Tag Number For A Log
===============================

This function is used to locate a tag number entered against a log number.

1. Choose Inventory Control » Physical Inventory » Find Tag Num For A Given Log [IN5L].

4. Enter or select Log Number.

The Tag Number assigned to the log number is displayed. If blank, the log number has not been assigned to a tag number (not counted or not entered in  in Inventory Count Data Entry).

5. Enter “Y” or / to exit.
