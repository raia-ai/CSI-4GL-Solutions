PO7W - Rebate Analysis Report
=============================

This report shows the rebates applied against receipt logs.

1. Choose Purchasing » PO Reports » Rebate Analysis Report [PO7W].
3. Enter or select the Vendor.
4. Enter or select the Rebate Type (defaults to all).
6. Select which Rebates to show.
