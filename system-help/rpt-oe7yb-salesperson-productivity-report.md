# OE7YB - Salesperson Productivity Report

1. Choose Sales Order Entry » Order Entry Reports » Salesperson Productivity Report \[OE7YB].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Select one or more Salesperson and enter the salesperson type (Inside or Outside).
4. Enter or select the Territory code (defaults to all).
5. Enter or select the Date.
6. Indicate if you want to Show salespeople with zero sales/quotes.
7. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
