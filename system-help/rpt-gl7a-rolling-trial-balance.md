# GL7A - Rolling Trial Balance - Excel

This function generates rolling trial balance report.

**NOTE:** This report is only available as an Excel download.

1. Choose General Ledger » GL Reports » Rolling Trial Balance - Excel \[GL7A].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Enter or select the Fiscal Period Start and End.
4. Enter or select General Ledger _Account From_ code (by default, all accounts are selected).
5. Enter or select General Ledger _Account To_ code (if all is selected in Account from field, this field is skipped).
6. Enter “Y” to display General Ledger accounts with balance zero (by default, these accounts are not shown).
7. Enter “Y” to display General Ledger accounts budget amounts (by default, these accounts are not shown).&#x20;
8. Report can only be downloaded to Excel using XLS printer code.
