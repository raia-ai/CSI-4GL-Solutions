OE7X - Sales By Customer By Product Line
========================================

This function produces a report containing sales value, weight quantity and gross margin percentage for each product line by customer within a given date range.

1. Choose Sales Order Entry » Order Entry Reports » Sales By Customer By Product Line [OE7X].
3. Enter or select a Customer code (by default, all are selected).
5. Indicate if you want to include Customer Groups and Product Line Details. If you want to Include Surcharges, enter “Y” if you want them to print on one line only per customer (if there are any), or “D” if you want surcharges to print individually on separate lines.
6. Indicate how the data should be Sorted: Alphabetically, lowest to highest sales Value, lowest to highest Weight, or lowest to highest Gross margin.
