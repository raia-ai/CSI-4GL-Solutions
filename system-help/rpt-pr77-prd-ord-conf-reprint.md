# PR77 - Production Confirmation

This function reprints a production order confirmation.

1. Choose In-House Production » In-House Production Reprints » Production Confirmation \[PR77].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select the Type of Entry.
4. Enter or select the Document Number.
5. Indicate if you want to Print Labels.
6. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
