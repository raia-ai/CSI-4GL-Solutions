# PR75 - Production Order

This function reprints a production order.

1. Choose In-House Production » In-House Production Reprints » Production Order \[PR75].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select the Type of Entry (by default, Production Order is selected “PR”).
4. Enter the original Document Number.
5. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
