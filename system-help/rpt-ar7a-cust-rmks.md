# AR7A - Displaying Remarks Assigned to Customers

This report displays the [remarks](https://4glsol.com/sm3-helpdocs/Content/Common%20Functions/Remarks.htm) assigned to customers.

1. Choose Accounts Receivable » AR Reports » Customer Remarks \[AR7A].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select the Customer code (defaults to ALL).
4. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
5. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
