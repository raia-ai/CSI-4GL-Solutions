# OE7YQ - Sales By Product Line By Vendor

This report lists sales by product line by vendor.

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Custom Reports » Sales By Product Line By Vendor \[OE7YQ].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Enter or select Vendor code.
4. Enter or select Product Line (defaults to all).
5. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
6. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
