Highlights
==========

Shipping Confirmation Columns can be Sorted
-------------------------------------------

The shipping confirmation defaults to sort by type of entry but you can click any column heading to sort by that column. 38057

OP Lot Header Cost
------------------

Hdr/Lot Costs button allows you to enter a header or “M” line lot charge to be distributed proportionally by weight across all finished goods lines in a set. 38140

Alternate Headings Forms have been Consolidated
-----------------------------------------------

The forms for Alternate Sales Order Headings (OE1S) and Alternate Invoice Headings (OE1Q) have been consolidated into an updated OE1Q, now called Alternate Form Headings, which allows alternate headings for the Packing List, Invoice, and Sales Order. 39067

Auto-Allocate Sets Window Added to PR41B
----------------------------------------

When creating an OP from PR41B you are now prompted to select the vendor, then the set allocation window is shown to allow you to auto-allocate sets. 38253

Kill Session from the Logged On Operators Screen
------------------------------------------------

If your operator is configured as a Super User (in MS32), you can now kill any session directly from the Logged On Operators screen. 38231

[this BG was hidden when Super User field was removed in 3.22]

Access Purch by Product from Detailed Inquiry
---------------------------------------------

When viewing the Top Vendors for a product in the Detailed Inquiry, a new Receipts button displays the same Purch by Product window as from the Vendors top menu. 38344

Address for Misc Docs on Manifest can be Changed
------------------------------------------------

The address on a new Misc doc in a manifest defaults to the customer ship-to but can be changed. 38352

Set a New Delivery Date When Approving a Backorder Review Task
--------------------------------------------------------------

When you approve a Backorder Review task, you can change the delivery date for one or more of the partially picked lines. If automatic assignment to manifests is enabled, the lines will be assigned to a manifest for the specified dates. 38403

Sales Order Inquiry can Search by RFQ#
--------------------------------------

An RFQ # field has been added to the search filters and results table in OE61. 38666

Manifest Inquiry
----------------

A new inquiry screen allows you to search and view manifests. 37709

Warehouse Sourcing for Customer Parts
-------------------------------------

When adding a line item, the Source field can now be set to “Vendor” or “Whse” for customer parts. 38645

Cash Receipts Can be Applied Automatically by Age or Particular Month
---------------------------------------------------------------------

Cash receipts can be automatically applied to customer invoices by Doc, Month, or Oldest first. 38432
