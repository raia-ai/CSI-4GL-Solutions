| ------ | --------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| 40260  | Remove GL37                                                     | Journal entries can no longer be deleted—a journal entry can be aborted if it has not yet been posted, and posted entries can be reversed.                                    |                                                                               |
