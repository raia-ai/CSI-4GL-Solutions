# Posting Journal Entries

Use this function to generate a proof listing of all, or selected, journal entries that have been processed, and then post them to the general ledger.

An authorization code is required; to obtain this code.

## Posting Regular Journal Entries

1. Choose General Ledger » Journal Entry » J/E Proof Listing / Update \[GL32].
2. Enter or select a singleJournal Number to post, or leave the default “ALL”.
3. If you selected “ALL” Journal Numbers, optionally enter or select an Operator code to filter only entries they created.
4. Click Print.
5. will generate the Proof Listing report, and then prompt you to Process Journal Entry Update. Enter the authorization code provided by 4GL Support.

## Posting Recurring Journal Entries

1. Choose General Ledger » Journal Entry » Recurring J/E Listing / Update \[GL36].
2. Indicate if you want the report to Sort by J(E) Or (G)L No.
3. Click Print.
4. will generate the Proof Listing report, and then prompt you to Process Recurring JE Update. Enter the authorization code provided by 4GL Support.
