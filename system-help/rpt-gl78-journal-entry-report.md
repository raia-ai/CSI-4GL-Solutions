# GL78 - Journal Entry Report

This function is used to print journal entry details for a selected fiscal period.

1. Choose General Ledger » GL Reports » Journal Entry Report \[GL78].
2. Enter or select Warehouse code (defaults to all warehouses).
3. Enter Document number, max seven alphanumeric characters.
4. Enter or select Accounting Period code (by default, all are selected).
5. Enter or select Start and End dates (by default, ending date is today's date).
6. Select Sub Account journal entries: Accounts Receivable, Accounts Payable, Accruals, Inventory, Other Sub Accounts = “Y” (by default, all are selected).
7. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
