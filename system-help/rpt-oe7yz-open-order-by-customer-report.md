# OE7YZ - Detailed Open Order By Customer

This function generates a report listing open orders sorted by customer.

1. Choose Sales Order Entry » Order Entry Reports » Detailed Open Order By Customer \[OE7YZ].
2. Select the Order status to view: ASG (not processed), PCK (picking confirmation), CRD (work order on credit hold) or All.
3. Indicate if Date Range should be based on the Ordered date or the Requested date.
4. Enter or select a Product Line (defaults to all).
