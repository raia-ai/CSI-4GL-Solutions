# Log Inquiry By Bin Location

This function is used to view log inquiry by bin location.

1. Choose Inventory Control » Inventory Inquiry » Log Inquiry By Bin Location \[IN6E].
2. Enter or select Location: Floor, Quarantine, Program, Kasto, Other or Blank for all. Select Warehouse Location.
3. Choose to Include Zero Quantity by entering “Y” or enter “/” to cancel.
4. Enter or select whether or not to include details.
