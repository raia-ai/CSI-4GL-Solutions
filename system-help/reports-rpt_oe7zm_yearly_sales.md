OE7ZM - Yearly Sales By Customer
================================

This report generates a report containing yearly sales and gross margin percent by customer on a rolling 12-year period.

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Custom Reports » Years Sales By Customer [OE7ZM].
3. Enter or select Customer Start Date (customers with a start date after this date will be included).
5. Enter or select a Territory code (by default, if “ALL” is selected the Province/State and Operator fields will default to “ALL”).
6. Enter or select a Province/State (by default, all are selected).
7. Indicate how you want to create the report by: Salesperson, Outside salesperson, or Order Writer.
8. Enter or select an Operator code (by default, all are selected).
9. Indicate if you want to use the Customer Salesperson or leave blank to use the salesperson on the invoices.
10. Indicate if you want the report to be Group by Operator.
11. Indicate how you want to Sort the data: by Customer name, Descending Sales, or Postal/zip code.
12. Indicate if you want to include customers with Zero Sales in the past year.
13. Indicate if you want to Show Details of the customers (postal code, phone #, contact, etc.).
14. Indicate if you want to Include Inactive customers.
