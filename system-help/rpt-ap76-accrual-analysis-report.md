AP76 - Accrual Analysis Report
==============================

This accrual report displays actual (from invoice) costs, estimated (from purchase order) and variance between the two costs, for each cost type, within a date range, for a selected or all buyers. This will also link return receipts with PO accruals.

1. Choose Accounts Payable » AP Reports » Accrual Analysis Report [AP76].
3. Enter or select Buyer code (by default, report is generated for all buyers).
5. Filter by Currency (by default, report is sorted by Vendor)
6. Enter “Y” to indicate whether to show only accruals that have zero actual value or leave default to show all.
7. Enter “Y” to indicate whether to show only accruals that have variance or leave default to show all.
8. Filter to create Summary or Detailed report.
