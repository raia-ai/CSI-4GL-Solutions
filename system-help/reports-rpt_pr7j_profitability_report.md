PR7J - Profitability Report
===========================

[needs description]

1. Choose In-House Production » In-House Production Reports » Profitability Report [PR7J].
3. Enter or select Type of Entry code: BT = branch transfer, CC = customer consignment, IV = sales invoice,  SO = sales order.
4. Enter or select the Document number.
5. Enter or select the Customer number.
