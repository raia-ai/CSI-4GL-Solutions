Splitting Existing Log Items
============================

This function is used to split (providing new dimensions) an existing Log item in inventory - option to return new dimension size to original log or create a new log.

1. Choose Inventory Control » Inventory Adjustment » Input Log Number [LULOGSP9].
3. Enter or select Log Number.

   1. Enter or select Product Line.
   2. Enter or select full or partial Product Code, or leave blank.
4. Press Save to process the function.
