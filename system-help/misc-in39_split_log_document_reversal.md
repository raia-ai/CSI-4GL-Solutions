Reversing Split Log Document
============================

This function is used to reverse Log Split created in LULOGSP9. Cannot reverse if Split logs created have been sold, picked or have been committed to a document.

1. Choose Inventory Control » Inventory Adjustment » Split Log Document Reversal [IN39].

4. Press F4 to view log Details.
5. Check box to have this piece returned to the Order.
6. Check to have this piece returned to original Log number.
7. Is this piece a scrap remnant enter “Y” for yes or leave blank for no in Scr field.
8. Press F4 to Revalue this piece of inventory.
