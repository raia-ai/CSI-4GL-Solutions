PO7G - Open PO By Vendor By Month (Wgt/Val)
===========================================

This function is used to display Open Purchase Orders by selected (or all) Vendor Types for a Warehouse, aging $ values by expected delivery dates.

1. Choose Purchasing » PO Reports » Open PO By Vendor By Month (Wgt/Val)[PO7G].
3. Enter or select the Aging Date (defaults to last day of current fiscal period).
4. Enter the Report Type: Product line or Vendor. If “V” is selected, Prod Line/By Grade fields are not available. If “P” is selected, the Vendor Type field is not available.
5. Enter or select the Vendor Type (defaults to all).
6. Enter or select Currency code (defaults to all). If you select a currency code, enter
7. Press F4 to select Product Lines, or leave blank to default to all.
8. Indicate if you want to sort/subtotal product lines By Grade.
9. Indicate if you want to show Details of purchase orders and pre-receipts .
