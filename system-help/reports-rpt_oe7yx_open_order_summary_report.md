OE7YX - Summary Open Order Report
=================================

This function generates a report listing open sales orders, and inventory lines on the sales orders.

1. Choose Sales Order Entry » Order Entry Reports » Summary Open Order Report [OE7YX].
3. Enter or select a product line group Pln Group (defaults to all).
