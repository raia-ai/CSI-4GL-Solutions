OE7R - Commission Report #1
===========================

This function is used to print an Inside Salesperson commission report. It will only list invoices paid within a date range, and reflects the Commission Rate assigned to the Inside Salesperson in OE13 (xref).

1. Choose Sales Order Entry » Order Entry Reports » Misc & Custom Reports » Misc Reports » Commission Report #1 [OE7R].
2. Enter or select a Salesperson code (defaults to all).
