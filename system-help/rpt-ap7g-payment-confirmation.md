# AP7G - Payment Confirmation

This function produces a list of paid checks, sorted by vendor, within a date range.

1. Choose Accounts Payable » AP Reports » Payment Confirmation \[AP7G].
2. Select a Vendor.
3. Enter number of the Check to confirm (by default, a range of checks within dates will be reported).
4. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
5. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
