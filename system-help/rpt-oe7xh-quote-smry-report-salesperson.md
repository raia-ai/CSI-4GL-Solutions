# OE7XH - Quote Summary By Salesperson

This function prints a summary report listing quotes by Inside/Outside Salesperson code.

1. Choose Sales Order Entry » Order Entry Reports » Quote Summary Report By Salesperson \[OE7XH].
2. Enter or select the Warehouse code and the warehouse type (Inventory or Sales).
3. Enter or select the Customer code (defaults to ALL).
4. Select one or more Salesperson and enter the salesperson type (Inside or Outside).
5. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
6. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
