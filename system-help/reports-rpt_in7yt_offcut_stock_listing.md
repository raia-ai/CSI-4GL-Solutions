IN7YT - Offcut Stock Listing
============================

This report displays the same columns as in IN71 by log and IN7ZG, but only displays reservations of logs from cut products.

1. Choose Inventory Control » Inventory Reports » Custom Reports » Offcut Stock Listing [IN7YT].
2. Enter or select Product code (by default, all are selected).


6. Select the Log Types to display: Reserved logs, logs Not reserved, or  All.
7. Enter or select the Product Line (by default, all are selected) and optionally, any further specifications.
