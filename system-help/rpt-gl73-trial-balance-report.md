# GL73 - Trial Balance

This function prints a trial balance.

1. Choose General Ledger » GL Reports » Trial Balance \[GL73].
2. Enter or select the Fiscal period code.
3. Enter the End Date for the fiscal period in (MM/DD/YY) format.
4. Enter “Y” to select this fiscal period (by default, the report is balanced to date).
5. Enter or select General Ledger _Account From_ code (by default, all accounts are selected).
6. Enter or select General Ledger _Account To_ code (if all is selected in Account from field, this field is skipped).
7. Enter “Y” to display General Ledger accounts with balance zero (by default, these accounts are not shown).
8. Select how/where the report should be output (click … to choose from a list). When you press Enter the report will generate.
