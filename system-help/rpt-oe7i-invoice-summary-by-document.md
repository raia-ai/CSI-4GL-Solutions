# OE7I - Invoice Summary By Day

This report generates a daily summary of all work orders which have been shipping confirmed within a given date range.

1. Choose Sales Order Entry » Order Entry Reports » Invoice Summary By Day \[OE7I].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Enter or select the Product Line Group (defaults to all).
4. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
5. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
