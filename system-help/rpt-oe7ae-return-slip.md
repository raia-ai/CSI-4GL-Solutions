OE7AE - Return Slip
===================

[need description here and in topic properties]

The Return Slips cannot be reprinted for credit memos in Approval status.

1. Choose Sales Order Entry » Order Entry Reprints » Return Slip [OE7AE].
3. Select the Type of Entry (“CM” or “DM”).
4. Enter or select the Document you want to print a return slip for.
