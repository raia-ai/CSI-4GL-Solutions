# OE7YI - Detailed Bookings Report

This function generates a report listing all open work orders and the remaining quantities to be shipped by order writer.

1. Choose Miscellaneous » OE Functions » Detailed Bookings Report \[OE7YI].
2. Enter or select Customer number (defaults to all).
3. Enter or select Salesperson code (defaults to all).
4. Enter or select Product line (defaults to all).
