IN7ZF - Product to Price Book Records Report
============================================

This function is used to display log reservations by product or product line(s).

This function is available as Excel download only.

1. Choose **Miscellaneous » IN Functions » Product to Price Book Records Report** [IN7ZF].
3. Enter or select *Product Line* code (by default, all are selected).
