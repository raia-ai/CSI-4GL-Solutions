# OE7A - Sales By Salesperson By Customer By Product Line

1. Choose Sales Order Entry » Order Entry Reports » Sales By Salespsn By Cust By Pline \[OE7A].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Indicate if you want to sort data by Operator (default) or Territory.
4. If Sort Order=“O”, select if the report should then be grouped by Salesperson or Order Writer.\
   If Sort Order=“T”, select the Territories to be included (in the window, indicate if you are Selecting or Deselecting territories.
5. Specify to create the report by Salesperson or Order Writer.
6. Enter or select Operator code (defaults to all).
7. Enter or select Customer number (defaults to all).
8. Enter or select Product Line code (defaults to all).
9. Enter the Dimension you want to report on.
10. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
11. Indicate if credit and debit memos should be referenced using the corresponding invoice date instead of credit/debit memo date.
12. Specify whether to create a Summary or Detailed (default) report.
13. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
