PO79 - Receipts By Vendor By Document
=====================================

This function is used to display receipts by vendor, by product lines for a selected time period. You have the option to filter by grade/dimension.

1. Choose Purchase Order » PO Reports » Receipts By Vendor By Document [PO79].
3. Enter or select Product Line code (defaults to all).
4. Enter the Grade and Finish (defaults to all).
5. Enter the Dimension From and Dimension To.
