# PO79 - Receipts By Vendor By Document

This function is used to display receipts by vendor, by product lines for a selected time period. You have the option to filter by grade/dimension.

1. Choose Purchase Order » PO Reports » Receipts By Vendor By Document \[PO79].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Enter or select Product Line code (defaults to all).
4. Enter the Grade and Finish (defaults to all).
5. Enter the Dimension From and Dimension To.
6. Select a Vendor.
7. Enter or select the starting date Date From and ending date Date To for the report (defaults to today's date).
8. Select whether to produce a Detailed or Summary report.
9. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
