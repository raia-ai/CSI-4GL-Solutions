Warehouse Min/Max Value Maintenance
===================================

This function is used to populate the warehouse inventory minimum and maximum values.

1. Choose Purchasing » Inventory Re-Order » Warehouse Min/Max Value Maintenance [PO19].
2. If you modify a line:
3. Enter Min and Max Inventory Units.
4. Enter Min and Max Pieces.
5. Enter or select default Vendor for this product warehouse.
