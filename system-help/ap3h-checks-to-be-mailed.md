Check Mailing List
==================

This function is used to view and update the mailed status of all printed checks. This is a powerful tool to use if checks are signed (and sometimes held) by different check signing authorities.

1. Choose Accounts Payable » Disbursements Processing » Checks To Be Mailed [AP3H].
2. Select whether or not M is. what is M?
3. Press F4 to enter remarks against this check, in the R field.
