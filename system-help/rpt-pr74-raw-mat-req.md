# PR74 - Raw Material Requirements

This function prints two reports: one showing the raw material requirements, and one showing the finished goods requirements.

1. Choose In-House Production » In-House Production Reports » Raw Material Requirements \[PR74].
2. Enter or select the Warehouse code (defaults to the warehouse assigned to you).
3. Press F4 to select the Finished Goods (products that have recipes defined in PR14).
4. Select how/where the report should be output (press F4 to choose from a list). When you press Enter the report will generate.
