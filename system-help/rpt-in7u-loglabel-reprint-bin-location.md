IN7U - Inventory Label By Bin Location
======================================

This function reprints log labels for all products within selected bin location(s), selected log number range. It requires a bar code printer installed on your network or attached to your computer.

1. Choose Inventory Control » Inventory Reprints » Inventory Label By Bin Location [IN7U].
3. Select Location Type: Kasto, Floor Stock (default), Quarantine Stock or Other.
4. Enter or select the Bin Location (by default, all bin locations are selected). If you go into the select screen, you can choose from all locations or enter a Location search filter to see only locations that match your filter.
5. Enter the Minimum and Maximum  log numbers (by default, all numbers are selected).
6. Indicate if you want to print logs with Available quantity only.
