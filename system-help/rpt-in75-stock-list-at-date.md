# IN75 - Stock List As At A Certain Date

This function produces a snapshot of the inventory stock list at a specified date.

1. Choose Inventory Control » Inventory Reports » Stock List At a Certain Date \[IN75].
2. Enter or select the Warehouse code plus _Child Warehouse_ indicator (defaults to warehouse assigned to operator).
3. Select one or more Product Lines.
4. Enter or select the Fiscal Period to report on.
5. Choose between a Summary or Detail report.
6. Select how/where the report should be output (click … to choose from a list). When you press Enter the report will generate.
