# Outstanding Check Listing

This function is used to print a list of outstanding checks.

1. Choose Accounts Payable » Check Reconciliation » Outstanding Check Listing \[AP63S].
2.  Enter All or Selective based on latest check date to be included in Type Of Report Required.

    If you select A, the following field will be skipped. If you select S then the following field will be open.
3. Enter latest Check Date to be included in outstanding check listing.
