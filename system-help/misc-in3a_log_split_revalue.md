Log Split Revaluation
=====================

This function is used to display all Log splits for a specified date.

1. Choose Inventory Control » Inventory Adjustment » Log Split Revalue [IN3A].
3. Enter From Date of which to display.
4. Enter Origin: HH = hand held, KA = kasto, blank = picking confirmation or “ALL”.
5. Enter Split Number.
